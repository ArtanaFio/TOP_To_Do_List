Example Todo List Item

Title: Prepare Presentation for Client Meeeting

Description: Learn best practices

Due Date: 8/15/2024

Priority: High

Notes:

  • Confirm the agenda with the team
  • Review the latest project metrics and include in the presentation
  • Allocate time for a Q&A session at the end

Checklist

  • Draft the outline of the presentation
  • Collect and update project data
  • Design slides with project updates and plans
  • Review slides with the team
  • Make final edits and polish the presentation
  • Send presentation to the client for review before the meeting
